# Relay Apps Registry

This directory contains the source of truth for all apps in the Relay Store.

## Contributing

1. **Add your app**: Create `apps/<bundleId>/manifest.json`.
2. **Submit PR**: Commit **ONLY** your `manifest.json` file.
3. **Do NOT commit `store.json`**: This file is automatically generated by GitHub Actions when your PR is merged.

## How to add a new app

1. Create a directory with your app's bundle ID (e.g., `com.example.myapp`).
2. Inside that directory, create a `manifest.json` file.
3. The `manifest.json` should follow the standard Relay App schema.

Example `manifest.json`:

```json
{
  "bundleId": "com.example.myapp",
  "name": "My App",
  "description": "Description of my app",
  "version": "1.0.0",
  "sizeBytes": 1024,
  "iconUrl": "https://...",
  "downloadUrl": "https://...",
  "screenshotUrls": [],
  "author": "Me",
  "category": "Utilities",
  "license": "MIT"
}
```

## Local Testing

If you want to verify your changes locally, you can run:

```bash
node scripts/build-store.cjs
```

But **discard changes to `store.json`** before committing.
