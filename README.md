## What are Clips?

Clips are lightweight apps (`.rly` bundles) that run on the Relay Shell. They're built with React and packaged with the Relay SDK.

## Browse Applications

See [Store](./store.json) for the full catalog.

## Installation

1. Open the Relay Shell
2. Go to the Clips Store
3. Tap "Get" on any app

Or via CLI:
```bash
relay install <bundleId>
```
